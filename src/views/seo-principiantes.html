Seo Básico - para principiantes


Crea títulos de página que sea únicos y atractivos


Usa los títulos de las páginas con las etiquetas title


  • La etiqueta < title > indica tanto a los motores de búsqueda como a los usuarios la temática de la web.

  • Esta etiqueta se debe colocar a su vez dentro de la etiqueta < head > en el documento HTML (1).

El contenido de la etiqueta title se muestra en los resultados de búsqueda


  • Si la página aparece en los resultados de búsqueda, el contenido de la etiqueta title, aparece en la primera línea de los resultados. Si son las mismas palabras que las que ha buscado el usuario, las palabras del título aparecerán destacadas en negrita. Esto ayudará a reconocer la relevancia de la página (2).


Usa un único título por cada página


  • El título de la página principal puede incluir el nombre del sitio web o negocio y podría incluir otro tipo de información relevante, como la dirección física de la empresa o sus principales actividades u ofertas (3).


Nota: La optimización de un sitio web para los buscadores afecta únicamente a los resultados de búsqueda orgánicos, no a los resultados pagados o patrocinados, como sería el caso de Google AdWords

Utiliza la metaetiqueta description


  • La metaetiqueta description de una página proporciona a Google y a otros motores de búsqueda un resumen sobre la página (1).

  • Mientras que el título de una página contiene unas pocas palabras, la metaetiqueta description podría contener un par de frases o incluso un párrafo corto.

  • Al igual que la etiqueta < title >, la metaetiqueta (1) La metaetiqueta description de nuestra página principal ofrece una breve reseña del sitio. description se coloca dentro de la etiqueta < head > en el documento HTML.

¿Cuáles son las ventajas de las etiquetas metadescripción?


  • Las metaetiquetas description son importantes porque Google podría utilizarlas como fragmentos de descripción de tus páginas. Ten en cuenta que decimos “podría” porque Google podría optar por utilizar una parte relevante del texto visible de tu página si ésta concuerda con la consulta del usuario.

  • Agregar metaetiquetas description para cada una de las páginas es siempre una buena práctica en caso de que Google no pueda encontrar un buen texto a utilizar como fragmento

Nota: Tener una metaetiqueta diferente para cada página ayuda tanto a los usuarios como a Google, especialmente en búsquedas en las que se obtiene las páginas indexadas de tu dominio (por ejemplo, búsquedas con el operador site:dominio => site:parodiaseo.com ) Esto dará como resultado todas las páginas o Urls que Google tiene indexado de esta web. Si tu sitio tiene miles o incluso millones de páginas, la elaboración de metaetiquetas description a mano no será factible. En este caso, Google puede generarlas automáticamente basándose en el contenido de cada página.

Recomendaciones

SI hacer

  • Crea etiquetas title únicas para cada página.


  • Elige un título que indique claramente el tema de la página.


  • Describe con precisión el contenido de la página


  • Lo ideal es que cada una de tus páginas tenga una etiqueta title única, que ayude a Google a distinguir esa página del resto de páginas de tu sitio.


  • Usa títulos descriptivos breves.


  • Los títulos pueden ser concisos pero informativos. Si el título es demasiado largo, Google mostrará tan solo una parte del mismo en el resultado de búsqueda.


  • Resume de forma precisa el contenido de la página

  • Utiliza descripciones únicas para cada página

NO

  • Un título que no tenga ninguna relación con el contenido de la página.


  • El uso de títulos predeterminados o demasiado genéricos como “Sin título” o “Página nueva”.


  • Títulos muy largos que no sean útiles para los usuarios.


  • Rellenar las etiquetas title con palabras clave innecesaria.


  • Una metaetiqueta description con contenido no relacionado con la página


  • Una descripción con sólo palabras clave


  • Copiar y pegar todo el contenido del documento en una metaetiqueta description


  • No Utilizar sólo una única metaetiqueta description en todas las páginas de tu sitio o en un gran grupo de páginas de tu sitio
